한국형 페르소나 데이터 추출기

원하는 특성을 선택해 Hugging Face 원본 데이터에서 최대 1,000명까지 수집하고, 페르소나 카드와 UX 리서치 산출물을 엑셀로 저장합니다.

데이터셋 대기 중

1. 원하는 특성 선택

조건이 좁을수록 수집 속도가 느려질 수 있습니다.

2. 수집 진행률

3. 원본 데이터 검색 결과

수집된 row의 일부를 미리 보여줍니다.

아직 수집된 데이터가 없습니다.

4. 페르소나 카드 / UX 리서치 산출물

원본 데이터 필드를 근거로 자동 구성합니다.

검색 후 페르소나 카드가 생성됩니다.

5. 데이터에 근거한 질문 답변

수집된 데이터 안에서만 요약합니다. 실제 사용자 조사 결과가 아니라 합성 데이터 기반 가설입니다.

검색 결과가 생성되면 질문 답변을 만들 수 있습니다.
